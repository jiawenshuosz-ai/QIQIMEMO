# 家庭共享备忘录

一个基于本地存储的家庭共享备忘录网页应用，支持多人使用、角色管理、深色模式和移动端适配。

## 功能特点

- 多人使用：添加、编辑、删除、标记完成备忘录
- 简单登录：首次使用需要注册账号密码，并选择角色（爸爸、妈妈、爷爷、奶奶、姥姥、姥爷、宝宝）
- 数据存储：使用浏览器本地存储，可通过导出/导入功能在不同设备间同步
- 界面简洁、手机适配、支持深色模式

## 技术栈

- 前端：HTML + CSS + JavaScript
- 数据存储：浏览器本地存储 (localStorage)

## 快速开始

### 1. 部署到阿里云

#### 方法1：使用阿里云对象存储OSS
1. **登录阿里云控制台**：访问 [阿里云官网](https://www.aliyun.com/) 并登录
2. **创建OSS Bucket**：
   - 进入对象存储OSS控制台
   - 点击"创建Bucket"
   - 填写Bucket名称，选择地域
   - 权限设置为"公共读"
3. **上传文件**：
   - 进入创建的Bucket
   - 点击"上传文件"
   - 选择所有项目文件（index.html, home.html, style.css, script.js, home.js, firebase-config.js）
4. **配置静态网站托管**：
   - 在Bucket概览页面，点击"静态网站托管"
   - 开启静态网站托管
   - 设置默认首页为"index.html"
   - 点击"保存"
5. **获取访问URL**：
   - 在静态网站托管页面，复制"访问域名"
   - 家人可以通过这个URL访问应用

#### 方法2：使用阿里云函数计算
1. **登录阿里云控制台**：访问 [阿里云官网](https://www.aliyun.com/) 并登录
2. **创建函数计算服务**：
   - 进入函数计算控制台
   - 点击"创建服务"
   - 填写服务名称，点击"确定"
3. **创建函数**：
   - 在服务下点击"创建函数"
   - 选择"HTTP函数"
   - 填写函数名称，点击"下一步"
   - 选择"空白模板"，点击"完成"
4. **上传代码**：
   - 在函数配置页面，点击"代码执行"
   - 将所有项目文件上传到函数代码目录
   - 保存并部署
5. **获取访问URL**：
   - 在函数详情页面，复制"触发URL"
   - 家人可以通过这个URL访问应用

## 使用说明

1. **首次使用**：
   - 打开部署后的URL
   - 点击"注册"，输入邮箱、密码并选择角色
   - 点击"注册"按钮完成注册

2. **登录**：
   - 输入注册时使用的邮箱和密码
   - 点击"登录"按钮

3. **管理备忘录**：
   - 在输入框中输入文本并点击"添加"按钮添加新备忘录
   - 点击"编辑"按钮修改备忘录内容
   - 点击"标记完成"按钮将备忘录标记为完成状态
   - 点击"删除"按钮删除备忘录

4. **同步数据**：
   - 点击"导出"按钮将备忘录导出为JSON文件
   - 将导出的JSON文件发送给其他家人
   - 其他家人点击"导入"按钮选择JSON文件导入备忘录

5. **其他功能**：
   - 点击右上角的"🌙"或"☀️"图标切换深色/浅色模式
   - 点击"退出登录"按钮退出当前账号

## 注意事项

- **数据存储**：数据存储在浏览器本地存储中，仅在当前设备上可用。如需在多个设备间同步，请使用导出/导入功能。
- **安全性**：密码以明文形式存储在本地存储中，仅适用于家庭内部使用。
- **数据备份**：建议定期导出备忘录数据作为备份。
